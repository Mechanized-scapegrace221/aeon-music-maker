## 🎛️ How to Generate Music

The application interface consists of a simple control panel. Follow this sequence to create your first track:

1. **Enter Your Prompt:** Locate the text box at the top of the interface. Type a detailed description of the music you wish to hear. For example, you might type "Upbeat acoustic folk song with guitar and light percussion." 
2. **Adjust Dynamics:** You will see a slider labeled "Mastering Level". This controls the loudness and tone of your output. For most projects, the middle position produces a balanced sound suitable for social media or personal listening.
3. **Set Length:** Select the duration of the audio clip using the dropdown menu. Shorter clips take less time to process.
4. **Initiate Generation:** Click the large button labeled "Generate". 
5. **Monitor Progress:** A progress bar appears at the bottom of the window. The application uses your computer's processor to build the audio file. Do not close the window while this bar moves.
6. **Save Result:** Once the generation finishes, an audio player appears. Press the "Play" button to verify the output. If you like the result, click "Export" to save the file as a high-quality song to your computer.

## 🧠 Understanding Mastering

Mastering is the final step in music production. It adjusts the volume and intensity of a track so it sounds professional across different devices. The aeon-music-maker tool uses automated digital signal processing to perform these tasks. By using the ACE Step 1.5 XL technology, the software keeps the musical dynamics intact. This means the quiet parts stay quiet, and the loud parts remain impactful without clipping or digital distortion. 

## ❓ Frequently Asked Questions

**Why does the application take time to start?**
The program loads several large model files into memory when it opens. This ensures that generations happen without errors.

**Can I run this on a laptop?**
You can run this on a laptop, but verify that your laptop has a dedicated graphics card. Systems relying solely on shared system memory for graphics will function, but generation will occur significantly slower.

**Where does the application store my songs?**
The program creates a folder called "exports" within the main application directory. Check this folder to find all your saved music files.

**How do I update the software?**
When a new version becomes available, revisit the main website. Download the latest release and extract the files into a new folder. You may delete the older folder once you verify that your settings are preserved or documented.

**Does the software need an internet connection?**
You need an internet connection to download the initial installer and the music models. Once you download these files, you can generate music offline. This protects your privacy and allows you to work without a steady network connection.

**What file formats does it support?**
The application saves files in the industry-standard WAV format. This format maintains full audio quality, allowing you to edit or convert the tracks in other software later.

**Is there a limit to the length of the track?**
To ensure hardware stability, the current version limits outputs to five minutes per generation. You can create multiple clips and combine them in any standard audio editor if you require a longer composition. 

**Are there settings for specific music genres?**
While you can type any genre into the prompt, the model is trained on a vast library of styles. Specificity in your prompt, such as "Lo-Fi Hip Hop," "Cinematic Orchestral," or "Electronic Dance Music," helps the underlying model choose the correct tempo and instrumentation for your request.
